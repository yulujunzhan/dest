bash简介

bash:unix shell,全称Bourne Again shell

是许多linux发行版的默认shell

bash常见命令

clear:清屏 键盘上箭头:自动回复

  1. pwd (Print Working Directory) 查看当前目录

  2. cd (Change Directory) 切换目录，如 cd /etc

     cd  切换目录
 cd .. 返回上一级
 cd ../.. 返回上一级的上一级
 cd 首字母+tab  自动补全,如果有多个匹配,会智能提示
 cd .  当前目录   ./也是当前目录

  3. ls (List) 查看当前目录下内容，如 ls -al

     ls -l 等同于-list
 ls -a 在 -l的基础上增加./ 和 ../
 ls -al/-la   -a和-l的合集
 ls 路径   查看其它文件夹下的内容

  4. mkdir (Make Directory) 创建目录，如 mkdir blog 

     创建多个文件夹  mkdir 1 2 3

  5. touch 创建文件，如 touch index.html

  6. cat 查看文件全部内容，如 cat index.html

  7. less 查看文件，如more /etc/passwd、less /etc/passwd

  8. rm (remove) 删除文件，如 rm index.html、rm -rf blog

     rm 1 2 3 4 :删除多个文件
 rm -rf  blog:删除文件夹和文件夹下的全部内容

  9. rmdir (Remove Directory) 删除文件夹，只能删除空文件夹，不常用

  10. mv (move) 移动文件或重命名，如 mv index.html ./demo/index.html

    mv 文件名 路径 :移动文件
mv 文件名 新文件名 :重命名

  11. cp (copy) 复制文件，cp index.html ./demo/index.html

    cp 文件名 路径 :复制文件到指定的目录
cp 文件名 路径+新文件名 :复制文件并重命名

  12. tab 自动补全，连按两次会将所有匹配内容显示出来

  13. 和 >>重定向，如echo hello world! > README.md，>覆盖 >>追加

    /> 将打印的结果追加到指定的文件并覆盖指定的文件的内容
/>>将输出的结果追加指定的文件文件

  14. | 管道符可以将多个命令连接使用，上一次（命令）的执行结果当成下一次（命令）的参数。

  15. grep 匹配内容，一般结合管道符使用
