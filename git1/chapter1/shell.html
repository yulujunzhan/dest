shell

  1. Shell 是一个用C语言编写的程序，它是用户使用Linux的桥梁。Shell既是一种命令语言，又是一种程序设计语言。

  2. shell是一种应用程序,该程序提供了一个界面,类似于DOS下的command和之后的cmd.exe,它接收用户命令,然后调用相应的应用程序

这里的shell不同于shell脚本

shell的分类

  1. 图形化界面的shell

    通过提供友好的可视化界面，调用相应应用程序，如windows系列操作系统，Linux系统上的图形化应用程序GNOME、KDE等。

  2. 命令行界面的shell

    通过键盘输入特定的命令,调用相应的程序,

    window下的cmd.exe, Window PowerShell , linux下的Bourne shell ( sh)、Bourne Again shell ( bash)

  3. 学习这部分课程主要了解的就是bash
