Git补充

1. 配置用户信息

a) Git config -l ----------------------------------------查看配置信息

b) Git config --global .....---------------------------配置本地信息,应用于所有的目录

c) Git config ......................--------------------------配置本目录信息

2. 文件的删除

a) Rm file---------------------------删除文件,不可恢复

b) Git rm file-----------------------删除文件,可以恢复,前提是文件已经提交到仓中,删除的是暂存区的文件

c) Rm -r file---------------------递归删除

d) Rm -f file---------------------规避错误信息,不显示错误的信息

e) Rm -rf file ------------------执行的是-r和-f的结合体

3. 删除本地仓库和远程仓库中的文件

a) 先删除缓存区的文件,再提交到仓库

b) 先删除工作目录中的文件,再逐步提交到仓库
